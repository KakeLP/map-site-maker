About the London Bookshop Map

This website originates from a Google map set up by Pedro and Kake to map all the bookshops in London. Eventually the number of bookshops on the map grew to the point where the Google Maps interface was no longer the best way to display the data.

The current site uses Perl, Template Toolkit, YAML, Leaflet, jQuery, MapQuest, and OpenStreetMap. It was built in emacs by Kake. Comments and questions to kake@earth.li.
